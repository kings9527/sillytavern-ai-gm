# 周六测试计划（06-13）

## 目标
经典完整克苏鲁单人跑团剧本测试，30-60 分钟体验

## 测试剧本
「密斯卡托尼克档案：消失的第七页」
（原创 5 场景完整剧本，Day 6 设计）

## 场景结构
1. 大学图书馆 → 发现古籍缺页
2. 教授办公室 → 调查学者背景
3. 阿卡姆公墓 → 夜间挖掘检定
4. 地下密室 → 邪教仪式现场（战斗/潜行分支）
5. 终局 → 理智检定决定结局（3 种结局）

## 完整要素
- 6-8 个 NPC（友好/中立/敌对）
- 10+ 个技能检定点（图书馆使用、侦查、聆听、说服、心理学、格斗等）
- 线索链：收集 3 个关键线索解锁最终场景
- 分支：战斗路线 vs 潜行路线
- 3 种结局：理智逃脱 / 精神崩溃 / 被邪教捕获
